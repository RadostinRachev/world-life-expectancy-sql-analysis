World Life Expectancy Analysis (SQL)
📌 Project Overview

This project focuses on cleaning and analyzing a global life expectancy dataset using SQL.
The goal was to identify trends, correlations, and insights related to life expectancy, GDP, BMI, mortality rates, and country development status.

The project follows a real-world data analyst workflow:

Data Cleaning
Data Validation
Exploratory Data Analysis (EDA)
Insight Generation

🧹 Data Cleaning Process

The data cleaning phase included:

Identifying and removing duplicate records
Standardizing inconsistent values
Handling missing life expectancy values
Updating incomplete status fields (Developed / Developing)
Improving overall data quality and consistency

📊 Exploratory Data Analysis

The analysis explored:

Life expectancy growth over time
Correlation between GDP and life expectancy
BMI impact on life expectancy
Adult mortality trends
Differences between developed and developing countries
Key SQL Concepts Used

🔍 Key Insights
Countries with higher GDP generally showed higher life expectancy
Developed countries had significantly higher average life expectancy
Adult mortality showed a strong negative relationship with life expectancy
Global life expectancy improved steadily over the analyzed period
